名字由来

关于名字尤尔小屋🏠的由来，取名来自于前段时间看的一本小说《尤尔小屋的猫》，于是我截取了前半部分

  • 该小说是英国新锐小说家——莉莉.海沃德Lili Hayward的作品，书中她有介绍到：$\color{red}{她是喜欢一切神秘而且有历史感的事物}$

  • 海沃德也是凭着《尤尔小屋的猫》一举成名，成为了英国近代文坛的奇迹。

  • 小说讲述的是一只猫和刚失恋的杰西之间的故事，很温暖，有爱，推荐指数⭐️⭐️⭐️⭐️

一切看似逝去的，都不曾离开，你所给与的爱与温暖，让我执着地守护着这里。

域名由来

博客的域名是www.renpeter.cn，是自行购买的。cn代表的是中国🇨🇳，是中国企业和个人的互联网标识。重点解释下renpeter😊

peter

peter很简单，就是自己一直以来都在使用的英文名，比较大众和俗气吧😢。但是在很多场合也用了很久啦😃​

ren

ren的拼音如果对应成汉字有很多：$\color{red}{人、忍、妊、仁、认、刃、韧……}$，在这里有3层含义：

  • 仁
  • 忍
  • 韧
仁

仁是我国古代的一种道德伦理观念。因为孔子对仁的重视和传播，逐步发展成了我国重要的道德标准，甚至上升到了一种哲学的境界。

引用来自百度百科的一段解释

从人从二，天地相爱有生即为“仁”字。因其有生故“仁”为种子内核中心，也是一切生命的中心

仁字可以说蕴含着中国人的三观，《系辞传》中解释到：

  • 阴阳有生即为“仁”：世界观
  • 以“仁”为参照是好、坏最初的起源：价值观。
  • 仁以为己任：人生观。

我理解的仁：应该是仁爱、感恩、善良

忍

关于忍字，最常听到的就是：

忍一时风平浪静，退一步海阔天空

这个字很有特点：在心的上面有一把锋利的刀刃。心里的有些话，或者想做的事情，不能说出来或者做出来，一直憋在心里，这便是忍。

很形象的一个汉字😃

我理解的忍：忍耐、忍受、克制💪

韧

韧字，本义指的是柔软而结实，受到外力作用时虽变形而不易折断。

我理解的韧：顽强持久、坚持不懈、死磕到底

大多数人的生活都不是一直风平浪静的。总会有或大或小的波动，遇到各种困境。我们能够做到的就是用一颗仁爱之心，忍受住生活的不幸遭遇，看做是一段经历就好，用坚韧的心态去乐观地生活。

感谢一位素未谋面的职场前辈送给我的三个字❤️

写博客 WHY

现在网上有很多的第三方平台都可以写博客，比如：

  • CSDN
  • 知乎
  • 掘金
  • 微信公众号
  • Github
  • …

为什么那么多的技术大咖都喜欢写博客？原因很多，具体请看这里➡️Look，总结一下大致原因如下：

  1. 能够交到志同道合的朋友
  2. 书写是为了更好地思考
  3. “教”是最好的“学”
  4. 讨论是绝佳的反思
  5. 激励自己持续地去学习和思考
  6. 学会持之以恒地做一件事
  7. 一个长期坚持的博客，是一份很好的简历

于是我也想拥有一个可以写作，可以自我发言，可以记录自己生活、工作、成长的平台。自然而然地，我也想建站了。

自建博客

第三方博客平台提供了一个便捷的平台，但总会有各种各样的限制：

  • 技术上：限制插件代码的使用，几乎不能自定义
  • 内容上：各种审查，谨防触雷🌩
  • 还有7788的广告，最烦人😓

自行建站就不同啦：我的地盘我做主，自由自在。在这里引用廖雪峰老师在知乎上的一段话：

自建博客+个人域名是树立个人品牌的一个重要步骤，试想别人通过搜索到你的个人网站的时候，说明个人品牌已经初步建立了。

😛 顺便给廖大神打了个广告
